# サポート / Support

BlinkGTK の配布パッケージに関する不具合・質問は、**本リポジトリの [Issues](https://github.com/daisy19gnu/blinkgtk-dist/issues) にご報告ください**。

Please report problems and questions about BlinkGTK **binary packages** via **[Issues](https://github.com/daisy19gnu/blinkgtk-dist/issues)** in this repository.

- バグ報告 / Bug report: 「バグ報告 / Bug report」テンプレートをご利用ください
- 質問・要望 / Questions & requests: 「質問・要望 / Question or request」テンプレート

報告前に [`INSTALL.md`](./INSTALL.md) のトラブルシューティングもご確認ください(依存不足・Wayland・glibc・sandbox など、よくある問題の切り分け方があります)。

Before filing, please also check the troubleshooting section of [`INSTALL.md`](./INSTALL.md).
